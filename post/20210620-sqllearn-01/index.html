数据库

DBMS

数据库，Database，实际上指的是数据内容本身，而我们要学习的是管理数据库，也就是要学习数据库管理系统，也就是所谓的DBMS，Database Management System

数据库是“按照数据结构来组织、存储和管理数据的仓库”。是一个长期存储在计算机内的、有组织的、可共享的、统一管理的大量数据的集合[1]

SQL，Structured Query Language 即结构化查询语言，是一种特殊目的的编程语言，是一种数据库查询和程序设计语言，用于存取数据以及查询、更新和管理关系数据库系统[2]

也就是说，我们学习SQL就是为了学习如何使用DBMS来更有效率的管理和操作我们的数据

DMBS的分类

根据其管理数据的方式可大致分为 关系数据库 和 非关系数据库，或称 NoSQL

这些概念我就直接引用百度百科了，大家也可以自己查询

关系数据库，是建立在关系数据库模型基础上的数据库，借助于集合代数等概念和方法来处理数据库中的数据，同时也是一个被组织成一组拥有正式描述性的表格，该形式的表格作用的实质是装载着数据项的特殊收集体，这些表格中的数据能以许多不同的方式被存取或重新召集而不需要重新组织数据库表格。关系数据库的定义造成元数据的一张表格或造成表格、列、范围和约束的正式描述。每个表格（有时被称为一个关系）包含用列表示的一个或更多的数据种类。 每行包含一个唯一的数据实体，这些数据是被列定义的种类。当创造一个关系数据库的时候，你能定义数据列的可能值的范围和可能应用于那个数据值的进一步约束。而SQL语言是标准用户和应用程序到关系数据库的接口。其优势是容易扩充，且在最初的数据库创造之后，一个新的数据种类能被添加而不需要修改所有的现有应用软件。主流的关系数据库有Oracle、DB2、SQLServer、Sybase、MySQL等[3]

NoSQL，泛指非关系型的数据库。随着互联网web2.0网站的兴起，传统的关系数据库在处理web2.0网站，特别是超大规模和高并发的SNS类型的web2.0纯动态网站已经显得力不从心，出现了很多难以克服的问题，而非关系型的数据库则由于其本身的特点得到了非常迅速的发展。NoSQL数据库的产生就是为了解决大规模数据集合多重数据种类带来的挑战，特别是大数据应用难题[4]

在这里，我们主要学习的是关系数据库，也就是用SQL语句管理的那些数据库管理系统

常见的DBMS

比较常见的和比较常用的关系数据库就是Oracle和MySQL以及PostgreSQL这三家了，由于MySQL并入Oracle公司，为了防止Oracle将原本开源的MySQL闭源，开源社区又从MySQL中fork出了MariaDB

以上这几个就是目前比较常用的关系数据库了

在嵌入式和安卓等移动终端上，还有个叫做SQLite的的轻型数据库比较常用，按分类，当属于关系数据库，不仅速度快，消耗还比其他数据库要小，正因如此，SQLite无法处理数据量较大的情况

NoSQL里面就数Redis和MongoDB比较出名，不过在日常中，还是关系数据库是主流，但是也在逐渐转向NoSQL了

之后如果有时间，关于SQLite和NoSQL我都会说一下使用

SQL语句

表，字段，观测

在关系数据库里，数据以表 (TABLE) 的形式组织起来，表的一行叫做一次观测，或者就称为数据 (data) 或记录，而表的一列则被称为一个字段 (column)，代表的是这个数据表中的一个特征或者分类，例如

学号  姓名  年龄
----------------
0001  张三   20
0002  李四   21

在这个表中0001 张三 20这样一行就被称为一次观测或者一条数据，是对于一个对象的一次全属性的记录，所谓属性就是说这个表中的字段，在这个表中，字段有学号, 姓名, 年龄这三个，代表所观测对象的某一特征或属性

由于现实因素，每个字段应该有这三个属性，即

  • 字段名
  • 数据类型
  • 相关约束

字段名很好理解，我就不多说了，相关约束就是对于这个字段的限制，比如学号和姓名不能为空，也就是必须填写，而年龄应该大于0岁小于256岁[5]等等

而数据类型则说明你应该填写什么，比如年龄，你应该填数字类型，而不能是像 而立 这样的字符串 (VARCHAR)

SQL语句分类

按照SQL语句的作用和使用范围，可以分为DQL，DML，DDL，TCL，DCL这五类

  • DQL: Date Query Language

    查询语句，凡是SELECT都是DQL

  • DML: Data Manipulation Language

    INSERT，DELETE，UPDATE等对表中数据的增删改

  • DDL: Date Definition Language

    CREATE，DROP，ALTER等对表结构的增删改

  • TCL: Transaction Control Language

    COMMIT提交事务，ROLLBACK回滚事务等

  • DCL: Data Control Language

    GRANT授权，REVOKE撤销权限等

创建数据库

要导入数据，首先要创建一个数据库来存储数据

对于 MariaDB 可以使用如下方式

1
2
3

create database <your-db-name>;

use <your-db-name>;

效果如下

MariaDB [(none)]> create database learnsql;
Query OK, 1 row affected (0.000 sec)

MariaDB [(none)]> use learnsql;
Database changed
MariaDB [learnsql]>

注意看，在使用use命令后，提示符的名称由(none)变成了learnsql，也就是我们创建的数据库的名称，这就意味着我们已经切换到我们创建好的数据库中，在MariaDB中，切换数据库可以使用use <db-name>;的方式来切换

PostgreSQL则已创建好了，我们使用psql连接进来后，我们的提示符learnpostgresql=#就说名我们已经在learnpostgresql这个数据库中了，当然，我们也可以在数据库中使用命令创建数据库，语法也是create database <db-name>，效果如下
